# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x     | Yes                |
| 1.x     | No                 |

## Reporting

PRISM is a theme collection with no executable code in production use. 

If you discover a security issue in the build tools or generators, please open a GitHub issue.

For sensitive disclosures, contact the maintainers directly.
