# PRISM - Where every color has purpose.

**64 curated color themes** built with WCAG AA verified contrast ratios.

## Features

- **64 curated themes** - Dark, light, and specialty variants
- **Visual effects** - Smooth caret, cursor glow, rainbow brackets
- **WCAG AA compliant** - Verified contrast ratios for accessibility
- **OLED optimized** - Safe black levels on dark themes

## Installation

1. Open **Extensions** (`Ctrl+Shift+X`)
2. Search **"Prism Themes"**
3. Click **Install**
4. `Ctrl+K Ctrl+T` to select a theme
