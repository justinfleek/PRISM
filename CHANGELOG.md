# Changelog

All notable changes to PRISM will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.0.0] - 2026-01-30

### Added
- 38 WCAG-verified color themes
- Multi-platform support: VSCode, Cursor, Neovim, Emacs, terminals
- OKLCH perceptual color space for mathematically accurate colors
- Formal verification proofs in Lean4
- Theme generator tools in Python and Haskell
- Terminal themes for Alacritty, Kitty, WezTerm, Windows Terminal, Helix, Starship

### Theme Collections
- **Core Prism (12)**: Tuned, Memphis, Github, Neoform, Biopic, Ghost, Tessier, Fleek Gold, Fleek Gradient, Blood Moon, Minimal, Rose Gold
- **Gradient Palettes (14)**: Coastal, Inferno, Faded Glory, Sunset, Forest, Ocean, Grape, Mint, Tropical, Arctic, Vesper, Vaporwave, Rose, Synthwave
- **Community Favorites (12)**: Acid Rain, Catppuccin Mocha, Cobalt2, Dracula Pro, Gruvbox Material, Holographic, Night Owl, Nord Aurora, One Dark Pro, Rose Pine, Synthwave 84, Ultraviolet

## [1.0.0] - 2026-01-26

### Added
- Initial release with 40 themes
- VSCode/Cursor extension
- Basic terminal theme support
